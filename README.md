# 🎯 Turbo Taskify

A personal task management app with pastel design built with Next.js (frontend) + Go (backend) + MongoDB.

## 📁 Project Structure

```
turbo-taskify/
├── frontend/     # Next.js React app (port 3000)
├── backend/      # Go REST API (port 8080)
└── ...
```

## 🚀 Quick Start

### Frontend

```bash
cd frontend
pnpm install
pnpm dev
# http://localhost:3000
```

### Backend

```bash
cd backend
go mod download
go run cmd/server/main.go
# http://localhost:8080
```
